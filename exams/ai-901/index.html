The exam

What is the AI-901 exam?

AI-901 is the current exam for the Microsoft Certified: Azure AI Fundamentals credential. It replaced AI-900, which retired on 30 June 2026. AI-901 is intended for people starting to work with AI solutions on Azure, including aspiring junior developers who are beginning to add AI capabilities to applications.

The current exam combines conceptual knowledge with foundational technical skills. You should understand core cloud concepts, Azure resources, common AI capabilities, responsible AI, and how client applications consume models and services. You also need to understand Python syntax and code examples that call AI models and services, and be able to use Microsoft Foundry to deploy models and implement single-agent solutions.

Frequently asked

AI-901 FAQs

How much does the AI-901 exam cost?

Microsoft prices AI-901 according to the country or region where the exam is proctored. Check the official scheduling page for the current local price before booking.

Does the AI-901 certification expire?

No. AI-901 earns the Microsoft Certified: Azure AI Fundamentals credential, and Microsoft Fundamentals certifications do not expire.

What is the AI-901 retake policy if I fail?

The first retake is allowed after 24 hours. Second and third retakes each require a 14-day wait. Microsoft caps retakes at five attempts per 12-month rolling period. Each attempt requires a new voucher purchase.

How long should I study for AI-901?

Allow several weeks for concepts plus hands-on practice. The current outline expects core cloud knowledge, familiarity with Azure resources, and an understanding of Python syntax and examples that call AI models and services. Azure Mastery's readiness gauge helps you decide when to schedule the exam.

AI-901 vs AI-900 — different exams?

AI-901 replaced AI-900 on 1 July 2026. AI-900 retired on 30 June 2026 and can no longer be scheduled. AI-901 remains the route to the Azure AI Fundamentals credential and adds a more implementation-oriented focus on Microsoft Foundry and Python examples.

AI-901 vs AI-102 — which next?

AI-901 is the fundamentals route. For a role-based next step, AI-103 covers developing AI apps and agents on Azure with Python and Microsoft Foundry. AI-102 retired on 30 June 2026.
